jsrsasign

The 'jsrsasign' (RSA-Sign JavaScript Library) is a open source free pure JavaScript implementation of PKCS#1 v2.1 RSASSA-PKCS1-v1_5 RSA signing and validation algorithm.

SOURCE CODES

The 'jsrsasign' library contains following source codes.

  • rsa-sign.js - RSAKey class extension for RSA signing and verification. (mini)
  • x509.js - X509 class to read subject public key from a PEM formatted X.509 certificate. (mini)
  • rsa-pem.js - RSAKey class extension to read PKCS#1 RSA private key PEM file (mini)
  • asn1hex.js - simple ASN.1 parser to read hexadecimal encoded ASN.1 DER (mini)
As for github repository, see https://github.com/kjur/jsrsasign.

How to sign

var rsa = new RSAKey();
rsa.readPrivateKeyFromPEMString(_PEM_PRIVATE_KEY_STRING_);
var hSig = rsa.signString("aaa", "sha1"); // sign a string "aaa" with key

How to verify signature

var x509 = new X509();
x509.readCertPEM(_PEM_X509CERT_STRING_);
var result = x509.subjectPublicKeyRSA.verifyString("aaa", _HEX_SIGNATURE_);

How to add supported signature algorithms

The script "rsa-sign.js" currently supports SHA1withRSA, SHA256withRSA, SHA512withRSA, MD5withRSA and RIPEMD160withRSA signature algorithms. However you can extend other signature algorithms such like MD2withRSA or SHA384withRSA by just specifying two variables in it. 
_RSASIGN_DIHEAD['md2'] = "30..."; // Hexadecimal DigestInfo prefix for MD5
_RSASIGN_HASHHEXFUNC['md2'] = md2hex; // function which returns value in hex.
