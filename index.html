FEATURES

The 'jsrsasign' library provides following features in pure JavaScript.
  • RSAKey - RSAKey extension for RSA signing and validation (PKCS#1 v2.1 RSASSA-PKCS1-v1_5)
  • Signature - digital signtature
  • MessageDigest - cryptographic hash calculation
  • ASN.1 encoder/generator
  • ASN.1 structure for X.509 ceritificate generation
  • simple ASN.1 data parser
  • simple X.509 certificate parser/reader
  • PKCS5PKEY - PEM formatted PKCS#5 password protected RSA private key

SOURCE CODES

The 'jsrsasign' library contains following source codes.

  • asn1.js - ASN.1 DER primitive encoder class. (mini)
  • asn1x509.js - ASN.1 structured for X.509 certificate. (mini)
  • crypto.js - MessageDigest and Signature class like Java JCE. (mini)
  • pkcs5pkey.js - encrypted PKCS#5 private key reader. (mini)
  • rsasign.js - RSAKey class extension for RSA signing and verification. (mini)
  • x509.js - X509 class to read subject public key from a PEM formatted X.509 certificate. (mini)
  • rsapem.js - RSAKey class extension to read PKCS#1 RSA private key PEM file (mini)
  • asn1hex.js - simple ASN.1 parser to read hexadecimal encoded ASN.1 DER (mini)
As for github repository, see https://github.com/kjur/jsrsasign.

INCLUDED THRID-PARTY LIBRARIES

  • Tom Wu's jsbn library - BigInteger and RSA
    • base64.js - String encoder for Base64 and Hex
    • jsbn.js - basic BigInteger class
    • jsbn2.js - BigInteger class extension
    • prng4.js - Random number generator
    • rng.js - Random number generator
    • rsa.js - RSAKey class for RSA public key encryption.
    • rsa2.js - RSA class extension for RSA private key decryption.
  • Paul Johnston's JavaScript MD5
    • You can download MD5, SHA1, SHA256, SHA512 and RIPEMD-160 implementation of JavaScript.
    NOTE: As for hash algorithm, you can use any other hash implementations. The requirement is to provide a function which returns hexadecimal string as the result.

NON INCLUDED THRID-PARTY LIBRARIES

  • CryptoJS - aes.js, tripledes.js, core.js, enc-base64.js, md5.js are used by 'pkcs5pkey.js'.
